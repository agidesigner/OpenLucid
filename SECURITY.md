# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.8.x   | :white_check_mark: |
| < 0.8   | :x:                |

## Reporting a Vulnerability

**Please do not open a public issue for security vulnerabilities.**

Instead, use [GitHub Security Advisories](https://github.com/agidesigner/OpenLucid/security/advisories/new) to report vulnerabilities privately.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

## Response Timeline

- **Acknowledgment**: within 48 hours
- **Initial assessment**: within 1 week
- **Fix or mitigation**: best effort, typically within 30 days for critical issues
